## Quick Start

1. Fork and clone the repository
2. Open in DevContainer (VS Code)
3. Run `pnpm install`
4. Configure `.env` file
5. Run `pnpm dev`

For detailed instructions, see [Developer Setup Guide](./developer-setup.md).
